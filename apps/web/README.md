# Web App

Primary D3HR product interface.

## Personas

- Employee
- Manager
- HR
- Institution Head
- Consultant

## First Screens

- Employee: daily goals, 3-line check-in, career progress, feedback
- Manager: team progress, blockers, feedback queue, evidence completeness
- HR: JobDB quality, approval queue, evidence readiness, Evidence Pack status
- Institution Head: aggregate HR risk briefing
- Consultant: project diagnostic board
