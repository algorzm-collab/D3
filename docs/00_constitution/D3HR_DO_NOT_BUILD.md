# Do Not Build

These items are intentionally out of scope until the constitution or roadmap changes through ADR.

- Full ERP replacement as the first market entry
- AI-only evaluation scoring
- AI-only placement decision
- Automatic low-performer labeling
- Payroll or compensation engine as MVP core
- Unlimited institution-specific customization
- Unmoderated anonymous community
- External evaluator access to draft or sensitive personal records
- Developer backdoor to production HR content
- Non-versioned job description edits
- Permission logic hardcoded per tenant
