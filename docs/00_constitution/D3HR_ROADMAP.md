# D3HR Roadmap

## v0.0-foundation

Goal: Establish governance, documentation, module boundaries, and GitHub configuration management.

Status: active baseline.

Deliverables:

- Product and development constitutions
- MVP scope
- Premortem
- Module charters
- Initial policies
- SOPs
- ADRs
- GitHub templates

## v0.1-mvp

Goal: Prove job-centered evidence loop.

Deliverables:

- Identity and tenant foundation
- Organization hierarchy
- Role assignment
- Policy evaluation foundation
- Audit logs
- NCS-to-JobDB seed model
- Job description version workflow
- Daily 3-line check-in
- OKR and evidence linkage
- Manager feedback
- HR Evidence Dashboard
- Basic admin

## v0.2-career

Goal: Turn evidence and job data into career-development value.

Deliverables:

- Career profile
- Job-fit scoring with explanations
- Competency gap
- Education recommendation
- Career progress bar
- Manager/HR career review workflow

## v0.3-community

Goal: Increase MAU through public-institution professional network.

Deliverables:

- Veteran profiles
- Job Q&A
- Best-practice library
- Benchmark-case publishing workflow
- Community moderation
- External network separated from internal HR data

## v1.0-pilot

Goal: Pilot-ready release for one public institution.

Deliverables:

- Stable MVP flows
- Tenant onboarding manual
- Access-control test matrix
- Audit-log verification
- Evidence Pack export
- Admin support workflows
- Release and rollback procedure
