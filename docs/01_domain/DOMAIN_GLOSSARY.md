# Domain Glossary

## Core Terms

- Tenant: A subscribed public institution.
- Organization: Internal org unit, such as division, department, team, or office.
- Job: Institution-level job definition derived from NCS and local practice.
- Task: Work activity performed within a job.
- Competency: Observable capability required for a job or task.
- Evidence: Work output, check-in, document, metric, feedback, or record supporting performance.
- Workflow State: The lifecycle state of a record, such as Draft, Submitted, Approved, Finalized, or Published.
- Access Policy: Rule combining role, org scope, sensitivity, state, cycle, and delegation.
- Evidence Pack: Auditable report package for management evaluation, audit, or internal governance.
- Veteran: Recognized job specialist with contribution, mentoring, or review authority.
