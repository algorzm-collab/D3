# Persona Map

## Employee

Daily reason to use D3HR:

- Check today's goals
- Record three-line work evidence
- See career progress
- Receive feedback
- Ask job questions

## Manager

Daily/weekly reason:

- See team progress
- Find blocked work
- Give feedback
- Review evidence
- Detect workload and performance risks

## HR

Daily reason:

- Operate JobDB
- Monitor evidence completeness
- Approve job-description changes
- Manage evaluation and policy workflows
- Produce Evidence Packs

## Institution Head

Periodic reason:

- View HR risk briefing
- Monitor job-centered HR transition
- Review workforce, performance, and management-evaluation risk

## Consultant

Project reason:

- Diagnose NCS mapping
- Review job-description quality
- Support policy and operating model design

## External Evaluator

Evaluation reason:

- Review finalized and submitted evidence packages only

## System Operator

Operational reason:

- Manage tenants, configuration, uptime, audit visibility, and support workflows without unrestricted HR-content access.
