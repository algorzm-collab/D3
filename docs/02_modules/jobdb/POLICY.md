# Policy: JobDB

## Rules

- Job descriptions are data, not static documents.
- Every finalized job description must have a version.
- NCS source mapping must be traceable.
- Local institution modifications must be distinguishable from NCS-derived content.
- Draft edits are not externally visible.
- Finalized versions can be superseded but not silently overwritten.

## Audit Events

- NCS import
- Job creation
- Version creation
- Approval
- Finalization
- Publication
- Archive
