# Module Charter: Work & OKR

## Purpose

Make D3HR a daily work hub by connecting goals, daily check-ins, work evidence, and job/task data.

## MVP Scope

- Daily 3-line check-in
- OKR cycle
- Objective and key result linkage
- Evidence attachment metadata
- Manager feedback hook

## Daily Loop

```text
Morning goal check -> work -> 3-line evidence -> manager feedback -> evidence accumulation -> career/performance update
```
