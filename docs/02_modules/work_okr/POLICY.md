# Policy: Work & OKR

## Rules

- Employees can create and edit their own draft check-ins.
- Managers can read submitted team check-ins within organization scope.
- HR can inspect evidence completeness but must respect sensitivity and workflow state.
- Institution heads see aggregates by default.
- System operators do not read restricted HR content by default.

## Audit Events

Audit logs are required for:

- Sensitive evidence read
- Evidence export
- Manager feedback on evaluation-relevant evidence
- Workflow transition into evaluation evidence
