# SOP: Work & OKR

## Employee Daily Check-In

1. Open daily home.
2. Review today's goals.
3. Write three-line work summary.
4. Link goal or task where available.
5. Add blocker or support request if needed.
6. Submit or save draft.

## Manager Review

1. Open team progress.
2. Review submitted check-ins within scope.
3. Respond to blockers.
4. Add feedback where needed.
5. Flag evidence gaps.

## HR Review

1. Monitor check-in and evidence completeness.
2. Identify teams or workflows with missing evidence.
3. Do not override records without proper workflow.
