# Access Control Policy

D3HR uses RBAC + ABAC + workflow-state authorization.

## Dimensions

- Role
- Organization scope
- Data sensitivity
- Workflow state
- Evaluation or OKR cycle
- Delegation
- Tenant configuration

## MVP Roles

- Employee
- Manager
- HR
- Institution Head
- Consultant
- System Operator

## Phase 2 Roles

- Evaluator
- Senior Manager
- External Evaluator
- Management Evaluation Group
- Developer Operator
- Veteran
- Community Moderator
