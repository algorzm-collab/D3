# AI Recommendation Policy

## Rule

AI may recommend. Humans and policy decide.

## Required Documentation For Any AI Recommendation

- Purpose
- Input data
- Excluded data
- Method or rule
- Explanation shown to users
- Bias or fairness risk
- Human approval path
- Audit-log event
- Prohibited uses

## Prohibited Uses

- Final evaluation score assignment
- Automatic low-performer labeling
- Final placement or promotion decision
- Hidden use of sensitive data outside policy
