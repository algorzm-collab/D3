# Tenant Customization Policy

## Principle

D3HR is a product, not unlimited custom consulting software.

## Allowed

- Tenant configuration
- Role assignment
- Workflow options
- Report templates
- Institution-specific job mappings
- Feature flags

## Not Allowed Without ADR

- Tenant-specific code branches
- Hardcoded permission exceptions
- Schema forks
- Custom UI logic that violates module contracts
