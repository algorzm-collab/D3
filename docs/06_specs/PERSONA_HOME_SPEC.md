# Persona Home Spec

## Purpose

Define first-screen product requirements so D3HR remains a daily-use product.

## Employee Home

Primary jobs:

- See today's goals.
- Submit 3-line check-in.
- See evidence accumulation.
- See career progress.
- See feedback.

Required data:

- active OKR cycle
- user's latest daily_checkin
- evidence count by week
- feedback notifications
- career progress summary

## Manager Home

Primary jobs:

- Review team progress.
- Respond to blockers.
- Check evidence gaps.
- Give feedback.

Required data:

- team submitted check-ins
- blocker count
- evidence gap count
- feedback queue
- workload signal summary

## HR Home

Primary jobs:

- Govern JobDB.
- Monitor evidence readiness.
- Manage approval queues.
- Prepare Evidence Pack.

Required data:

- JobDB completeness
- job-version approval queue
- check-in/evidence completion rate
- audit exception count
- Evidence Pack readiness

## Institution Head Home

Primary jobs:

- Read HR risk briefing.
- Track job-centered HR transition.
- See workforce and performance risk.

Required data:

- aggregate evidence readiness
- workload imbalance summary
- job-fit risk summary
- transition index
- management evaluation risk

## Consultant Home

Primary jobs:

- Diagnose NCS mapping.
- Review job-description quality.
- Track project progress.

Required data:

- NCS mapping rate
- job-description review queue
- missing competency/task mappings
- project-scoped issue list
