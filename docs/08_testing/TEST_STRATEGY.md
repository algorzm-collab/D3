# Test Strategy

## Test Categories

- Unit tests
- Integration tests
- Permission tests
- Workflow-state tests
- Audit-log tests
- Tenant-isolation tests
- Data-model migration tests
- UI role-visibility tests

## MVP Acceptance

- Employee can create daily check-in.
- Manager can review scoped team evidence.
- HR can manage job-description workflow.
- Institution head sees aggregate briefing only.
- System operator cannot silently view unrestricted sensitive HR content.
- Sensitive actions create audit logs.
